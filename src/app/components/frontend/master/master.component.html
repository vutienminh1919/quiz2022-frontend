HTML

Với người mới bắt đầu lập trình web thì HTML là một trong những ngôn ngữ cơ bản đầu tiên mà bạn cần phải nắm vững. Các bài quiz của chúng tôi sẽ giúp bạn

CSS

CSS là chữ viết tắt của Cascading Style Sheets, nó là một ngôn ngữ được sử dụng để tìm và định dạng lại các phần tử được tạo ra bởi các ngôn ngữ đánh dấu (HTML). Nói ngắn gọn hơn là ngôn ngữ tạo phong cách cho trang web. Các bài quiz của chúng tôi sẽ giúp bạn ( trừ ai tên Hoàng )

JavaScript

Là 1 trong bộ tam tấu của trang web cùng CSS và HTML, JavaScript là ngôn ngữ lập trình phổ biến nhất trên thế giới trong suốt 20 năm qua. Các bài quiz của chúng tôi sẽ giúp bạn

PHP

PHP là một ngôn ngữ lập trình kịch bản hay một loại mã lệnh chủ yếu được dùng để phát triển các ứng dụng viết cho máy chủ, mã nguồn mở, dùng cho mục đích tổng quát. Nó rất thích hợp với web và có thể dễ dàng nhúng vào trang HTML. Các bài quiz của chúng tôi sẽ giúp bạn

LARAVEL

Laravel là một khung web PHP mã nguồn mở, miễn phí, được tạo bởi Taylor Otwell và dành cho việc phát triển các ứng dụng web theo mô hình kiến trúc của bộ điều khiển xem view mô hình và dựa trên Symfony. Các bài quiz của chúng tôi sẽ giúp bạn
